## 功能描述

- 核心IC：stm32f103

- 基础支持：

>2个串口：1个下载，一个用于他机通信
>
>2套I2C：【SCL+SDA+GND】
>
>2套SPI：【SCK+MISO+MOSI+CS】
>
>2套5V电源：[5 G S]
>
>2套3V3电源：[3V3 G S]

- 板载支持

> 显示器件
>
> >4位数码管
> >8路led
> >
> >OLED12864(串并可选)？
>
> 输入器件[非传感器]
>
> >16按键
>
> 输出器件
>
> >振动电机(pwm)
>
> 传感器[I2C、SPI为主]
>
> >温度传感器[I2C]
>
> 存储器件
>
> >8G MicroSD卡槽[SDIO+SPI]
> >
> >256M eeprom
